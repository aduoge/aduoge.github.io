首先是关于这个博客的Markdown形式介绍

技巧如下

在最开始，这里介绍一下关于如何设置阅读全文的方法：
第一点是如上面所说，”“这个东西
第二个方法是，在next的config文件中，找到这个
auto_excerpt:
enable: false
length: 150
可以将这个false修改为true，然后length表示的是这个可见长度

1.“>”就是起到区块引用的功能
2.“#”的个数表示的是标题的重要性，一个是主标题
需要注意的是，在这个“#”后面一定要加一个空格，不然不会显示标题分级
同时也可以用“*”来替换，因为着两个一直可以互相使用

3.关于字体
1.首先是粗体，粗体采用“粗体”这个格式
2.斜体，“斜体”，注意的是这个和—是等价的
4.这个—三个就表示分割线，或者**
5.关于序列
1.如果是无序的话，则用 -第一个项目
-第二个项目
2.如果是有序的话，则用数字加”.”比如1.2.3.4.
6.链接（最难的部分）
1.行内式
基于一起的链接，将链接基于文字，点击文字就可以点开链接
我的链接
2.参考式
[我的链接1][1]
[我的链接2][2]
[1]:www.yaohelove@qq.com
[2]:www.18698668022@163.com
上面的链接在实际中是隐藏的，但是这个文字后面代表的链接
7.’’这两个点会起到标记的作用，比如’cnt+a’在实际效果中就是加红的
