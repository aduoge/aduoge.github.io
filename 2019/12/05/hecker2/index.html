IP

  1. IP地址采用的是点分十进制方法，中间使用“，”来间隔开，分为4段，每段包括8位

  1. 组成：网络标识码和主机标识码。

  2. IP地址划分为五类，ABCDE

  3. IP地址如果全0，则代表当前网络，全1则代表广播地址

查询IP地址方法

  1. 最简单就是百度搜索”IP地址查询”，本机IP地址为211.140.198.197

  2. 如果计算机是连接路由器实现上网的话，则会拥有两个IP，一个是外网IP，一个是内网IP（使用ipconfig查看）

黑客专用通道—–端口

TCP/IP协议提供socket接口

  • socket接口

    可以让计算机通过软件的方式与任意一台具有socket应用程序进行通信，计算机中可开启的端口号数值范围是1-65535

    端口的分类

    1. 公认端口

      通常包括0-1023号端口，绑定了一些特定的服务，例如80号端口分配给HTTP协议使用，因此无法重定义80端口

    2. 注册端口

      没有明确的作用对象

    3. 动态和/或私有端口

  • 关闭端口

进程：程序在系统中的一次执行过程

  1. 进程分类

    • 系统进程：用于完成系统的各种功能的进程 常用作黑客设置，使这个系统中的木马或者病毒与系统进程名称十分相似
    • 程序进程：通过启动应用程序所产生的进程

  2. 关闭系统进程 explorer.exe
    在windows10上名字为 windows资源管理器，就会让桌面消失，只剩下输入设备

  3. Ping命令
    主要用于确定本地计算机于另一台计算机额能否交换数据包，常用的指令如下：

    1. -a:将输入的IP地址解析为计算机NetBIOS名。（Network Basic Input Output System位于会话层的API接口）
      1. -n count: 可以自己定义发送的个数
      2. -t： 一直Ping指定的计算机

  4. IPconfig命令：用来查看TCP/IP的简单配置信息
